Tools

Decode the artifacts by hand.

Paste a token, a SAML message, or a certificate and see exactly what's inside — signature, claims, conditions, keys. No profile, no round-trip to an identity provider.

Tokens

JWT decoder → Decode header and claims, read the timestamps, and verify the signature against a JWKS or shared secret. JWKS viewer → List the keys in a JWK Set — kid, type, algorithm, use, and RFC 7638 thumbprint — from JSON or a URL.

SAML

Response decoder → Base64 → XML, with NameID, attributes, and signature, audience, and time-condition checks. AuthnRequest decoder → Inflate a Redirect-binding request or decode a POST one, and read its issuer, ACS, and requested context.

Encoding

Encode / Decode → Base64, Base64URL, URL escaping, DEFLATE, and JSON/XML formatting — the plumbing behind every protocol.

Keys & certificates

Certificates & keys → Inspect an X.509 certificate — subject, validity, fingerprints — or generate a fresh SP keypair.
